## Como Rodar Localmente

Como é um projeto estático, você pode abrir o `index.html` direto no navegador.

Se preferir servir por HTTP local:

```bash
python3 -m http.server 8000
```

Depois acesse: `http://localhost:8000`

## Como Contribuir

1. Faça um fork do repositório.
2. Crie uma branch para sua alteração:
3. Envie a branch e abra um Pull Request descrevendo:

- o que mudou;
- por que mudou;
- como validar.

## Boas Práticas para PR

- Não alterar estrutura principal da página sem necessidade.
- Manter acessibilidade básica (`alt`, `aria-label`, contraste).
- Evitar remover conteúdo existente sem justificativa.
- Preferir mudanças pequenas em vez de PRs muito grandes.

## Sugestões de Tipos de Contribuição

- Correções de texto e ortografia.
- Novos contéudos ou complementar oque já existe.
- Melhorias visuais e responsivas.
- Acessibilidade e semântica HTML.
- Refatoração de JS/CSS sem mudar comportamento.
- Novas referências técnicas confiáveis.
